Напишите функцию

struct FileContent
{
    ssize_t size;
    char *data;
};
struct FileContent read_file(int fd);
Функция должна считать все содержимое файла по переданному файловому дескриптору.
В случае ошибки чтения поле size должно быть равно -1, а поле data — NULL.
В случае успешного чтения поле size должно содержать размер считанного файла,
поле data — указатель на считанные данные. Область памяти data
должна быть на 1 больше размера файла, и последний байт в области должен содержать
байт 0. Таким образом, текстовый файл в памяти будет хранится как строка (null-terminated string).
Ситуация нехватки памяти должна обрабатываться. В этом случае возвращаемое значение
такое же, как при ошибке чтения.
Для буферизации ввода используйте локальный буфер размера 4096 байт.
Используйте POSIX API работы с файлами.
Помните, что read не обязан всегда считывать ровно то количество байт,
которое было запрошено.
При сдаче программы структура FileContent будет уже определена.
Не определяйте ее в сдаваемом на проверку коде.
