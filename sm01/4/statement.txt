Дано описание структуры:
struct Pair
{
    int key;
    int value;
};
Напишите функцию с прототипом:
void process(struct Pair *data, size_t size);
Параметр data указывает на начало массива из size элементов.
Функция process должна отсортировать массив data в порядке неубывания значения поля key.
Элементы с равным значением поля key должны сохранить относительный порядок следования в массиве.
На проверку сдается только описание структуры struct Pair и реализация функции process.
Функция main должна отсутствовать. Необходимые заголовочные файлы должны быть включены.
При тестировании значение size не превосходит 1000000.
