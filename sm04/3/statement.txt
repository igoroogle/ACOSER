Дан следующий перечислимый тип:

typedef enum FPClass
{
    FFP_ZERO, FFP_DENORMALIZED, FFP_NORMALIZED, FFP_INF, FFP_NAN
} FPClass;


Напишите следующую функцию:


FPClass fpclassf(float value, int *psign);

Функция на вход принимает число value типа float и классифицирует его.
Если число не равно NAN, то в переменную psign возвращается знак числа (1 для отрицательных чисел).
Для чисел NAN знак всегда должен быть 0.

При сдаче функции на проверку должно присутствовать только тело функции. Ни определение перечислимого типа FPClass,
ни функция main присутствовать не должна.
Запрещается использовать стандартные функции и расширения gcc для
классификации чисел.
